  2. /events
    1. /create - POST

      Create new event

      Role = 1

    2. /update - POST

      Updates event

      Role = 1

      Changes can do only user who organise event (ORGANISER_ID)

    3. /getAll - GET

      Return array of all events

      Role = 2

    4. /getByCategory - POST

      Returns array of events by CATEGORY

      Role = 2

    5. /getByName - POST

      Returns array of events by NAME

      Role = 2

    6. /getByPlace - POST

      Returns array of events by PLACE

      Role = 2

    7. /getByStart - POST

      Returns array of events by START

      Role = 2

    8. /remove - POST

      Returns array of events by START

      Role = 1

      Remove can do only user who organise event (ORGANISER_ID)

  3. /clubs
    1. /create - POST

      Creates new club

      Role = 2

    2. /update - POST

      Updates club

      Role = 2

      Changes can do only user who owns club (USER_ID)

    3. /getAll - GET

      Returns array of clubs

      Role = 2

    4. /getByUser - POST

      Returns array of clubs by USER_ID

      Role = 2

    5. /remove - POST

      Remove club

      Role = 2

      Remove can do only user who owns club (USER_ID)

  4. /jb
    1. /create - POST

      Creates new jb

      Role = 1

    2. /createFromCvs - POST

      Creates new jb from .cvs file and returns fileName

      Role = 1

    3. /createByCvs - POST

      Creates new jb from .cvs fileName that has been send by /createFromCvs

      Role = 1

    4. /update - POST

      Updates jb

      Role = 2

      Changes can do only user who owns club (CLUB_ID -> USER_ID)

    5. /getByClub - GET

      Returns array of jbs by club

      Role = 2

      Can only get club owner (CLUB_ID -> USER_ID)

    6. /remove - POST

      Remove jb

      Role = 2

      Remove can do only user who owns club (CLUB_UD -> USER_ID)
